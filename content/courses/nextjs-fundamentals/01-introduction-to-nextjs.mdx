# Introduction à Next.js

Next.js est un framework React qui permet de créer des applications web complètes, du frontend au backend.

## Qu'est-ce que Next.js ?

Next.js est un framework React qui offre des fonctionnalités comme le rendu côté serveur (SSR), la génération de sites statiques (SSG), et bien plus encore.

## Pourquoi Next.js ?

- **Performance** : Optimisations automatiques
- **SEO** : Rendu côté serveur pour un meilleur référencement
- **Expérience développeur** : Rechargement à chaud, routage simplifié
