# Tour du Propriétaire

Explorons ensemble la structure d'un projet Next.js et comprenons le rôle de chaque fichier et dossier.

## Structure de base

- `app/` : Contient les routes et les composants de votre application
- `public/` : Fichiers statiques accessibles directement
- `next.config.js` : Configuration de Next.js
