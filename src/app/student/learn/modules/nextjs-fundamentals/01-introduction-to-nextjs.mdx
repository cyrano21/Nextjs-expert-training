# Introduction à Next.js : Votre Première Application Web Moderne

<InfoTip>
Next.js est un framework React puissant qui simplifie le développement d'applications web full-stack avec un rendu côté serveur et client.
</InfoTip>

## Commencer un Nouveau Projet Next.js

<TerminalOutput>
npx create-next-app@latest mon-premier-projet
cd mon-premier-projet
npm run dev
</TerminalOutput>

<ProjectStep title="Configuration Initiale">
Créons un nouveau projet Next.js avec la dernière version. La commande ci-dessus :
1. Crée un nouveau projet
2. Se déplace dans le dossier du projet
3. Lance le serveur de développement
</ProjectStep>
